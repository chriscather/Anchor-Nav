Anchor Nav

Nav for On-Page Anchors

Download this project as a .zip file Download this project as a tar.gz file

Dynamic

  1. Smooth scroll to each section with no anchor tags in your address bar.
  2. Navigation "Active" item will follow each section as you scroll down.
  3. The "Next" button on the bottom right will always take you to the next anchor.
  4. In the last section, the "Next" button automatically becomes a "Back to Top" button.

Fluid Responsive

This menu system is fully responsive. Sticky and evenly spaced menu items without reliance on "Flex-Box", which is not supported in all versions of IE.

jQuery

This menu system is compatible with jQuery libraries back to 1.7. If your website is already using jQuery, than you only need to add a few lines of javascript to make this menu system work.
